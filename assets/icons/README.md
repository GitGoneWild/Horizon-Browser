# Horizon Browser Icons

This directory contains the icon assets for the Horizon Browser.

## Required Icons

- `horizon-logo.png` - Main application logo (256x256)
- `horizon-icon.ico` - Windows icon
- `horizon-icon.icns` - macOS icon
- `tab-icon.png` - Default favicon for tabs (32x32)

## Icon Guidelines

- Use SVG format for scalable icons when possible
- Provide multiple resolutions for raster icons
- Follow the dark theme color palette
- Maintain consistent visual style across all icons
