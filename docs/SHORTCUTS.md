# Horizon Browser Keyboard Shortcuts

Complete guide to keyboard shortcuts in Horizon Browser.

## Tab Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+T` | New Tab | Opens a new tab with the home page |
| `Ctrl+W` | Close Tab | Closes the current tab (won't close last tab) |
| Click Tab | Switch Tab | Switches to the clicked tab |
| Click ✕ | Close Tab | Closes the specific tab |

> **Note**: On macOS, use `Cmd` instead of `Ctrl`

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Alt+Left` | Go Back | Navigate to the previous page in history |
| `Alt+Right` | Go Forward | Navigate to the next page in history |
| `Ctrl+R` | Reload | Reloads the current page |
| `F5` | Reload | Reloads the current page (alternative) |
| `Alt+Home` | Go Home | Navigate to your homepage |
| `Ctrl+L` | Focus Address Bar | Selects the address bar for typing |

## Address Bar

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Enter` | Navigate | Navigate to the entered URL or search |
| `Ctrl+L` | Select All | Focus and select address bar content |

These URLs can be entered in the address bar:

| URL | Description |
|-----|-------------|
| `about:home` | Opens the browser home page |
| `about:blank` | Opens a blank page |

## Tips & Tricks

### Quick Navigation
- Use `Alt+Left/Right` to quickly move through your browsing history
- Press `Ctrl+L` to quickly type a new URL without clicking the address bar

### Tab Management
- Open multiple tabs with `Ctrl+T` and switch between them easily
- Close tabs you don't need with `Ctrl+W` to keep things organized

### Search Shortcuts
- Type search terms directly in the address bar - no need to visit a search engine
- The browser will automatically detect if you're entering a URL or search query

### Efficient Reloading
- Use `F5` or `Ctrl+R` to refresh the current page
- The reload button also acts as a stop button when a page is loading

## Platform-Specific Notes

### Windows & Linux
- Use `Ctrl` for all shortcuts
- `Alt` key for navigation shortcuts (Back, Forward, Home)

### macOS
- Use `Cmd` (⌘) instead of `Ctrl` for most shortcuts
- `Option` (⌥) instead of `Alt` for navigation shortcuts
- Examples:
  - New Tab: `Cmd+T`
  - Close Tab: `Cmd+W`
  - Back: `Option+Left`
  - Forward: `Option+Right`

## Accessibility

### Screen Readers
- Navigate through UI elements using Tab key
- Activate buttons with Space or Enter

### Keyboard-Only Navigation
- All major functions are accessible via keyboard
- Tab through controls in order
- Use arrow keys in dropdown menus
